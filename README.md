# jack_delay (Nix Package)

Nix package for `jack_delay`, a tool to measure the round-trip latency of a soundcard.

## Build Instructions

### Classic Mode
Uses `flake-compat` to build via `nix-build`.
```bash
nix-build
```

### Flake Mode
```bash
nix build
```
